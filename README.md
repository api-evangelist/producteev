# ProducteevThis is a repo for managing machine readable artifacts for the [Producteev APIs](https://www.producteev.com/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Producteev APIs](https://www.producteev.com/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://www.producteev.com/)- [Developer](https://www.producteev.com/)- [Twitter](https://twitter.com/producteev)- [GitHub](https://github.com/Producteev)The [APIs.json](https://github.com/api-evangelist/producteev/blob/master/apis.json) index provides links to all the relevant links for the Producteev APIs, including the OpenAPI and Postman if they are present.
